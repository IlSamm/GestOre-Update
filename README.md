# GestOre Update Channel

Benvenuto nel canale ufficiale degli aggiornamenti di **GestOre Desktop**.

Questa repository pubblica è dedicata esclusivamente alla distribuzione delle **release ufficiali** dell’app, così da permettere agli utenti di scaricare e installare sempre la versione più recente in modo semplice, rapido e automatico.

## Cosa contiene questa repository

Qui vengono pubblicati soltanto i file necessari per gli aggiornamenti e per l’installer di **GestOre Desktop**:

- `GestOre-Setup-x.x.x.exe`
- `GestOre-Setup-x.x.x.exe.blockmap`
- `latest.yml`

## A cosa serve

Questo canale serve per garantire una distribuzione degli aggiornamenti più veloce ed efficiente, permettendo all’app di ricevere nuove versioni senza dover scaricare manualmente ogni volta l’intero pacchetto in modo più lungo e scomodo.

## Cos’è GestOre

**GestOre** è un’app desktop progettata per offrire un’esperienza semplice, moderna e pratica nella gestione delle attività e dell’organizzazione quotidiana.  
L’obiettivo dell’app è quello di rendere tutto più immediato, ordinato e accessibile, con un’interfaccia chiara e un sistema in continua evoluzione.

L’app è attualmente ancora in fase di sviluppo attivo, ma ha già raggiunto un ottimo punto sotto il profilo delle funzionalità, della struttura e dell’esperienza generale.  
Proprio per questo continuerà a ricevere aggiornamenti costanti, miglioramenti e nuove funzioni nel tempo.
